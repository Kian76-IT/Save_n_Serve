# save_n_serve

A new Flutter project.

Save n Serve is a Flutter-based mobile app that aims to help reduce food waste by connecting those with excess food with those in need. The app provides a digital platform that simplifies the food donation and distribution process, ensuring that edible food can be optimally utilized rather than wasted.
The app supports two types of users: Givers, who donate food, and Recipients, who receive donations. Through Save n Serve, users can create and manage food donations, view a list of available donations, save donations to a watchlist, connect activity history, and receive notifications regarding donation progress.
By combining mobile technology and social responsibility, Save n Serve is expected to be a solution that helps reduce food waste while increasing access to food for those in need.

## ✨ Features
- User Authentication (Sign In & Sign Up)
- Role-based User (Giver & Beneficiary)
- Food Donation Management
- Food Listing & Detail View
- Watchlist
- Activity History
- Notifications
- User Profile Management
